# Pre-Launch Checklist

## Complete this checklist before launching your marketing campaigns

---

## Landing Pages

- [ ] Landing pages are hosted and live
- [ ] URLs are working (test on desktop, tablet, mobile)
- [ ] All forms are functional
- [ ] Form submissions are being received (test submission)
- [ ] Thank you messages display correctly
- [ ] All links work
- [ ] Images load properly
- [ ] Mobile-responsive design works
- [ ] Fast loading time (< 3 seconds)
- [ ] No broken links or errors

---

## Form Integration

- [ ] Form handler is set up (EmailJS, Formspree, etc.)
- [ ] Test form submission sends email
- [ ] Email is received in correct inbox
- [ ] Form data is captured correctly
- [ ] Spam protection is working (if applicable)
- [ ] Form validation works (required fields)
- [ ] Error messages display correctly

---

## Email Automation

- [ ] Email platform is set up (Mailchimp, ConvertKit, HubSpot)
- [ ] Email templates are imported and customised
- [ ] Automation sequence is created (7 emails)
- [ ] Test: Form submission triggers welcome email
- [ ] Email delays are set correctly (Day 2, 5, 8, 12, 18, 30)
- [ ] Merge tags work ([Name], [Suburb], etc.)
- [ ] Email design looks good on mobile
- [ ] Unsubscribe link works
- [ ] All links in emails work

---

## CRM Setup

- [ ] CRM is set up (Google Sheets or HubSpot)
- [ ] All columns/fields are created
- [ ] Form connects to CRM (test entry appears)
- [ ] Follow-up process is documented
- [ ] Mel & Jeff know how to use CRM
- [ ] Notification system works (if applicable)

---

## Tracking & Analytics

- [ ] Google Analytics 4 is set up
- [ ] Tracking code is on all landing pages
- [ ] Test: Visit landing page shows in Google Analytics
- [ ] Conversion events are set up (form submission)
- [ ] Google Search Console is set up (if applicable)
- [ ] Facebook Pixel is installed
- [ ] Test: Facebook Pixel fires on page load
- [ ] Google Ads conversion tracking is set up (if launching ads)
- [ ] All tracking codes are working

---

## Google Business Profile

- [ ] Profile is claimed/created
- [ ] All information is complete and accurate
- [ ] Business hours are correct
- [ ] Phone number is correct (Australian format)
- [ ] Address is correct
- [ ] Service areas are listed (Drouin, Trafalgar, Warragul, etc.)
- [ ] Description includes keywords
- [ ] 10+ photos are uploaded
- [ ] Logo is set as profile photo
- [ ] First Google Post is created

---

## Australian Platform Profiles

- [ ] RealEstate.com.au agent profile is created/optimised
- [ ] Domain.com.au agent profile is created/optimised
- [ ] Professional photos are uploaded
- [ ] Bio highlights Gippsland expertise
- [ ] Service areas are listed
- [ ] Contact information is correct
- [ ] Links to listings (if applicable)

---

## Ad Accounts (If Launching Ads)

### Google Ads
- [ ] Account is created
- [ ] Currency is set to AUD
- [ ] Location is set to Gippsland, Victoria, Australia
- [ ] Billing is set up (if ready to launch)
- [ ] First campaign is created (but not launched yet)
- [ ] Ad copy is written
- [ ] Landing page URLs are correct
- [ ] Conversion tracking is set up
- [ ] Budget is set

### Facebook Ads
- [ ] Facebook Business Manager is set up
- [ ] Facebook Page is created/optimised
- [ ] Facebook Pixel is installed and tested
- [ ] Ad account is created
- [ ] First ad set is created (but not launched yet)
- [ ] Ad creative is prepared
- [ ] Landing page URLs are correct
- [ ] Budget is set

---

## Content

- [ ] First blog post is written (if applicable)
- [ ] Social media content is prepared (2 weeks worth)
- [ ] Ad creative is prepared (images, videos)
- [ ] Testimonials are collected
- [ ] Recent sales data is available (for market updates)

---

## Team Preparation

- [ ] Mel & Jeff are briefed on follow-up process
- [ ] CRM system is explained to team
- [ ] Response time expectations are set (within 24 hours, ideally 1 hour)
- [ ] Follow-up scripts are provided
- [ ] Team knows how to access leads
- [ ] Team knows how to update CRM
- [ ] Appraisal process is documented

---

## Legal & Compliance

- [ ] Privacy policy is accessible (if required)
- [ ] Terms of service are clear (if applicable)
- [ ] Email opt-in is clear (for email automation)
- [ ] Australian consumer law compliance (if applicable)
- [ ] Real estate licensing requirements met

---

## Testing

### End-to-End Test
- [ ] Complete test: Ad click → Landing page → Form submission → Email received → CRM entry → Follow-up
- [ ] Test on different devices (desktop, tablet, mobile)
- [ ] Test on different browsers (Chrome, Safari, Firefox)
- [ ] Test form submission multiple times
- [ ] Verify all tracking is working

### Performance Test
- [ ] Landing page loads in < 3 seconds
- [ ] Images are optimised (not too large)
- [ ] Mobile experience is smooth
- [ ] No broken functionality

---

## Budget & Financial

- [ ] Monthly ad budget is determined
- [ ] Daily ad budget is calculated
- [ ] Payment method is set up (for ads)
- [ ] Budget limits are set (to prevent overspending)
- [ ] Cost per lead targets are set
- [ ] ROI goals are defined

---

## Monitoring Setup

- [ ] Analytics dashboard is created (Google Sheets or similar)
- [ ] Daily check-in routine is scheduled
- [ ] Weekly review process is set up
- [ ] Key metrics are defined:
  - Cost per click
  - Cost per lead
  - Cost per appraisal
  - Conversion rates
  - ROI

---

## Documentation

- [ ] All passwords are saved securely
- [ ] Account access is documented
- [ ] Process documentation is complete
- [ ] Team has access to necessary guides
- [ ] Troubleshooting guide is available

---

## Final Checks

- [ ] All items above are completed
- [ ] Team is ready to respond to leads
- [ ] You're ready to launch!
- [ ] Launch date is scheduled
- [ ] Monitoring schedule is set

---

## Launch Day

### Morning:
- [ ] Final review of all systems
- [ ] Launch Google Ads (start with small budget)
- [ ] Launch Facebook Ads (start with small budget)
- [ ] Verify ads are showing
- [ ] Monitor first few hours closely

### Afternoon:
- [ ] Check for first leads
- [ ] Verify tracking is working
- [ ] Monitor ad performance
- [ ] Adjust if needed (but give it time to optimise)

---

## Post-Launch (First Week)

### Daily:
- [ ] Check for new leads
- [ ] Monitor ad performance
- [ ] Check ad spend vs. budget
- [ ] Review search terms (add negative keywords)
- [ ] Respond to any issues

### End of Week:
- [ ] Review week's performance
- [ ] Calculate cost per lead
- [ ] Identify what's working
- [ ] Identify what needs improvement
- [ ] Plan adjustments for next week

---

## Success Criteria

**Week 1 Goals:**
- Ads are running and showing
- Landing pages are receiving traffic
- Form submissions are coming in
- Leads are being followed up
- Tracking is working correctly

**Month 1 Goals:**
- Consistent lead flow
- Cost per lead within target ($40-150 AUD)
- 10-20 appraisals scheduled
- System is running smoothly

---

## Troubleshooting

If something isn't working:
1. Check the specific guide for that topic
2. Test the specific functionality
3. Check for error messages
4. Verify all settings are correct
5. Contact support for the specific tool (if needed)

---

## Notes

- Don't try to launch everything at once
- Start with Property Appraisals campaign first (highest priority)
- Give ads time to optimise (at least 7 days before making major changes)
- Monitor closely but be patient
- Focus on getting the system working, then optimise

---

*Complete this checklist before launching. It's better to delay launch by a day to fix issues than to launch with problems.*
